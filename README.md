매일 학습한 내용을 기록하는 개인 학습 저장소입니다.

## 📁 구조

### daily-logs/
일일 학습 기록을 날짜별로 저장합니다.
- 형식: `YYYY/MM/YYYY-MM-DD.md`
- 예시: `2025/06/2025-06-10.md`

### categories/
주제별로 학습 내용을 정리합니다.
- `java/`: Java 관련 학습 내용
- `data-structures/`: 자료구조 학습 내용
- `algorithms/`: 알고리즘 학습 내용

### templates/
일일 기록 및 정리용 템플릿을 저장합니다.
